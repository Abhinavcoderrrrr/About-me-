Skills

HTML (HyperText Markup Language) is the standard language used to create and design websites and web applications. It provides the basic structure for web content, such as text, images, links, and other media.

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of an HTML document. It controls the layout, colors, fonts, spacing, and other visual elements on a webpage. CSS allows you to separate the content (HTML) from the presentation (styling), making web design more flexible and maintainable.

JavaScript (JS) is a high-level, dynamic programming language commonly used in web development. It allows developers to create interactive and dynamic content on websites, such as form validation, animations, interactive maps, and much more. JavaScript is often used in conjunction with HTML and CSS to create modern web applications.

Node.js is a powerful, open-source, cross-platform runtime environment that allows developers to run JavaScript on the server-side. It was built on the V8 JavaScript engine (the same engine that powers Google Chrome) and allows for building scalable, high-performance applications. With Node.js, you can use JavaScript both on the client-side (in the browser) and the server-side (on the backend).

MongoDB is a NoSQL (Not Only SQL) database management system that stores data in a flexible, JSON-like format known as BSON (Binary JSON). It is a document-oriented database, which means that instead of storing data in rows and columns like traditional relational databases (e.g., MySQL, PostgreSQL), MongoDB stores data as documents, making it highly flexible and scalable.
